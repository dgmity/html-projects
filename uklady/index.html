Flexbox

Flexbox (Flexible Box Layout) to sposób na łatwe rozmieszczanie elementów w jednej osi (poziomej lub pionowej), z pełną kontrolą nad ich rozmieszczeniem, wyrównaniem, oraz przestrzenią pomiędzy nimi.

Grid

CSS Grid Layout to zaawansowane narzędzie do tworzenia dwuwymiarowych układów, które pozwala na precyzyjne ustawianie elementów zarówno w poziomie, jak i pionie.

Układ za pomocą tabel

Stosowanie tabel w CSS do tworzenia układów jest rzadziej używane w nowoczesnym web designie, ale może być przydatne w sytuacjach, gdzie wymagany jest precyzyjny układ w formie wierszy i kolumn. Stosując właściwość table-layout, możemy łatwo kontrolować sposób wyświetlania zawartości tabeli.

Inline-block

Metoda inline-block łączy cechy zarówno elementów inline (czyli ustawiających się obok siebie), jak i block-level (które zajmują całą dostępną szerokość). Dzięki temu elementy mogą być ustawione w jednym wierszu, ale zachowują cechy blokowe, takie jak możliwość nadawania rozmiarów (szerokości i wysokości).

Multi column layout

CSS pozwala na tworzenie układów wielokolumnowych przy użyciu właściwości column-count, column-gap. Jest to przydatne do tworzenia układów przypominających gazetę lub magazyn.
